# 1. 分级基金 #

**1.1 折溢价计算**

	*  收盘后计算评估 
- 
	*	场内合并价 =（A收盘价*A占比 + B收盘价*B占比）
	*	场内溢价 = 场内合并价 / [ 母基净值 * ( 1+申购费率）] 
 
	*  盘中动态评估
- 
	*	场内合并价 =（A买一 *A占比 + B收买一*B占比）
	*	母基净值估值 = 母基前一日净值 *（1 + 指数涨幅*95%）
	*	场内溢价 = 场内合并价 / [ 母基净值估值 * ( 1+申购费率）] - 1
